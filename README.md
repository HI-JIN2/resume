# Resume

개인 이력서를 웹으로 배포하기 위한 Vite + React 프로젝트입니다. 안드로이드 개발자용과 일반 개발자용 이력서를 한 화면에서 전환할 수 있게 구성되어 있습니다.

## 환경

- Node.js 18 이상
- npm 9 이상 (또는 호환되는 패키지 매니저)

## 설치

```bash
npm install
```

## 개발 서버 실행

```bash
npm run dev
```

## 프로덕션 빌드

```bash
npm run build
```

`dist/` 폴더에 정적 파일이 생성되며, `npm run preview`로 빌드 결과를 로컬에서 확인할 수 있습니다.
